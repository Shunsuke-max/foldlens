# FoldLens Image Gen prompt set

Generated with the built-in Image Gen tool in `ui-mockup` mode.

## Desktop concept

Design the complete primary desktop analysis workspace for a local-first AlphaFold 3 results app named “FoldLens”, for structural and computational biologists. Show a quiet header with the FoldLens brand, “KRAS · SOS1 complex”, local-only privacy status, Open result, and Export. Use four regions in a 1440×900, no-scroll screen: five predictions on the left with Model 1 selected and scores 0.92/0.89/0.84/0.81/0.76; a large uninterrupted central 3D molecular canvas with confidence/chains/surface/reset controls and Ranking/ipTM/pTM/clash metrics; a narrow right inspector with interpretation, chain visibility, and chain-pair confidence; and a bottom PAE heatmap linked to 3D selection. Visual direction: dark graphite/navy scientific instrument, warm white type, restrained cyan/lime accents, thin separators, precise compact data typography, modest radii, almost no shadow, consistent 1.5px outline icons. Keep all app text and controls code-native and practical for React + Vite. Avoid marketing UI, card grids, glassmorphism, purple gradients, cyberpunk glow, decorative filler, fake business metrics, browser/device frames, and unreadable microtext.

## Mobile concept

Create the 430×932 responsive counterpart of the approved FoldLens desktop workspace with the same graphite/navy, cyan/lime, typography, separators, and instrument-like restraint. Use a single-column task flow rather than compressed desktop sidebars: compact header; large interactive molecular viewer; Ranking 0.92, ipTM 0.88, pTM 0.91, and No clashes; Structure/PAE/Models/Insights tabs; then the selected Structure content with Interpretation and visible A/B/L chain controls; finish with Open another result. Keep the viewer dominant, controls at least 44px, content vertically scrollable and readable, all text code-native, and component families reusable from desktop. Avoid marketing treatment, card grids, glassmorphism, neon glow, device frames, or tiny desktop panels.
